# Examen Tema 1 - API Canvas 2D

## Descripción
Aplicación web desarrollada con HTML, CSS, Bootstrap y JavaScript utilizando la API Canvas 2D para dibujar una fresa mediante programación.

## Características
- Más de 30 figuras básicas
- Código organizado por funciones
- Comparación con imagen original
- Diseño responsivo con Bootstrap
- Navbar y Footer personalizados
